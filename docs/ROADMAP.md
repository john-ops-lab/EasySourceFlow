# 路线图

本路线图按版本管理。已发布版本使用确定日期；规划版本使用目标时间，可能根据平台接口变化和真实使用反馈调整。

## v0.1.1 - 安全检查通用化（已发布，2026-07-15）

- 将仓库路径和发布配置检查改为通用匹配规则。
- 删除安全测试中与特定开发环境绑定的检查值。

## v0.1.0 - 首个公开版本（已发布，2026-07-15）

- 完成 HTTP 服务、Web 控制台、stdio MCP 和 OpenClaw 接入。
- 支持普通网页、微信公众号、Bilibili、YouTube 实验能力及本地文档。
- 完成平台字幕优先、本地 ASR 回退、来源标记和核心要点时间轴。
- 完成 OpenAI-compatible 模型配置、Fast/Pro 选择和通用提示词管理。
- 完成 SQLite 持久化任务、重启恢复、缓存、全文搜索、资源包和收藏夹。
- 完成 LaunchAgent、自检、日志、备份、清理、维护通知和安全发布流程。

详细变更见 [更新日志](../CHANGELOG.md)。

## v0.2.0 - YouTube 正式支持（已发布，2026-07-15）

- 支持从本机 Chrome 安全导入和检测 YouTube 登录态。
- 建立人工字幕、原语言自动字幕、其他字幕和本地 ASR 的明确优先级。
- 区分登录、PO Token、限流和字幕下载失败状态。
- 增加 YouTube 单元、HTTP、浏览器和真实平台回归入口。

## v0.3.0 - Web 音视频下载（已发布，2026-07-15）

- 增加仅限 Web 的 Bilibili/YouTube 单视频和音频下载。
- 下载任务支持持久化、进度、取消、重试、恢复和附件交付。
- 升级到 Python 3.10+、当前 `yt-dlp[default]` 和 YouTube EJS 挑战求解链路。
- 保持下载能力与 Agent/MCP 工具隔离。

## v0.4.0 - 模型生态扩展（已发布，2026-07-15）

- 增加 MiniMax、Gemini、硅基流动、Ollama、LM Studio、xAI、豆包、百度千帆和腾讯混元。
- 支持 Chat Completions 与豆包 Responses API 返回结构。
- 模型 ID 可选择预设或手动输入，本地回环模型不强制要求 API Key。

## v0.5.0 - 登录与模型输出可靠性（已发布，2026-07-16）

- Bilibili 和 YouTube 登录后自动接入 Chrome 登录态，并支持从 EasySourceFlow 退出。
- 验证平台认证 Cookie，避免匿名 Cookie 被误判为登录成功。
- 修复 Bilibili 扫码导入误报以及运行时同步恢复旧登录状态的问题。
- 模型测试兼容推理输出、Chat Completions 和 Responses API。
- MiniMax、DeepSeek 及其他兼容模型统一只交付最终总结，不展示推理过程或提示词。

## v0.6.0 - 配置体验与代理兼容（已发布，2026-07-16）

- 重做模型配置与选择流程，按服务商独立保存、切换和删除 API Key。
- 结果库改为新页面打开，任务完成后立即提供结果入口。
- 增加 Web Fake-IP 可信网段配置，同时保留严格 SSRF 防护。
- 对普通网页重定向目标重新执行 URL 安全校验。
- 补齐安装、首次配置、Fake-IP 自检和常见故障排查文档。
- 扩展桌面端、移动端、模型切换和网络设置的浏览器回归测试。

## v0.6.1 - 首次配置提示修正（已发布，2026-07-16）

- 示例模型 API Key 默认留空，首次启动会准确提示用户配置真实凭据。
- 区分阻塞错误和未登录平台账号的提醒，避免普通网页用户误判服务不可用。

## v0.6.2 - Bilibili 字幕身份与时长校验（已发布，2026-07-16）

- 使用精确 BVID、CID 和分 P 的 WBI 接口选择 Bilibili 字幕，移除旧播放器接口回退。
- 在总结前拒绝时间倒序、超出视频时长或覆盖严重不足的字幕，失败后转入本地 ASR。
- 结果包记录可审计的字幕来源信息，并通过缓存版本升级淘汰旧错误结果。
- 增加真实字幕错配回归样例，检查 BVID、字幕来源和字幕时长比。

## v0.6.3 - Bilibili 短链接字幕修复（已发布，2026-07-16）

- 从 `yt-dlp` 已验证的页面地址恢复短链接对应的 BVID 和分 P，再调用 WBI 字幕接口。
- 保留原始短链接作为来源，并清理结果中不必要的分享会话参数。
- 增加短链接平台字幕优先、禁止无谓 ASR 和失败路径诊断回归测试。

## v0.6.4 - Agent 原样交付修复（已发布，2026-07-17）

- 修正飞书卡片对象类型和失败回退规则，禁止 Agent 因卡片发送失败自行压缩总结。
- OpenClaw Skill 白名单用户会在安装输出和接入文档中收到显式启用提醒。
- 增加完整 Markdown 卡片交付和失败回退回归场景。

## v0.7.0 - 视频处理扩展（计划：2026 年第四季度）

- 评估并接入 FunASR 或其他中文 ASR 后端。
- 支持 Bilibili 多 P 视频。
- 持续适配 YouTube 客户端和 PO Token 规则变化。

## 后续评估（暂无版本承诺）

- 本地菜单栏入口和更直接的服务状态操作。
- 真实 Bilibili 失败样例库、ASR 质量基准和恢复任务输出幂等性。
- Obsidian vault、模板和 Local REST API 可选接入。
- NotebookLM、RAG、本地向量索引和统一知识库整理。

## 明确不做

- 公网多用户服务和权限系统。
- 绕过 DRM、付费墙或平台权限限制。
- 面向批量归档或规避平台限制的专业视频下载器。
